# NexusHub — Modular Personal Workspace

**NexusHub** — это модульное SPA-приложение для организации цифровой жизни. В отличие от стандартных "хабов" с фиксированной структурой, NexusHub позволяет пользователю создавать **собственные Страницы** и наполнять их виджетами (Категориями) разного типа: от заметок и вики до музыкального плеера и закладок.

Проект построен на стеке (React 18 + Vite), использует **Supabase** в качестве Backend-as-a-Service и **Cloudinary** для хранения медиафайлов.

---

## 🚀 Ключевые Возможности

### 🏗️ Динамическая Архитектура
- **Конструктор страниц:** Пользователь сам создает разделы (например: "Работа", "Хобби", "Музыка").
- **Универсальные категории:** Внутри каждой страницы можно создавать блоки разного типа:
  - 🔗 **Ссылки:** Визуальные закладки и ссылки браузера.
  - 📝 **Карточки:** Гибрид заметок и вики-статей.
  - 🎵 **Аудио:** Музыкальные плейлисты.
  - 🎲 **Генераторы:** Рандомайзеры решений.

### 📝 Продвинутый Редактор Карточек (Card Editor)
Редактор на базе **TipTap** с поддержкой блочной структуры:
- **Smart Media:** Загрузка изображений, видео (MP4/YouTube) и аудио.
- **Галереи:** Карусели изображений с подписями.
- **Связи (Backlinks):** Вставка ссылок на другие карточки в виде интерактивных каруселей.
- **Полиморфизм:** Карточка может быть простой текстовой заметкой или полноценной 3D-статьей с обложкой.

### 🎵 Глобальный Плеер
- Воспроизведение музыки фоном при навигации по сайту.
- Поддержка плейлистов, drag'n'drop сортировки треков.

### 🏠 Домашняя Страница (Dashboard)
- Агрегатор контента со всех страниц пользователя.
- Виджеты: "Случайная заметка", "Быстрые ссылки", "Случайный трек", "Генератор дня".

## 🛠 Технический Стек

**Frontend:**
- **Core:** React 18, TypeScript, Vite.
- **UI:** Chakra UI, Framer Motion (анимации), React Icons.
- **State:** Zustand.
- **DnD:** `@dnd-kit` (сенсорное управление, сортировка списков и сеток).
- **Editor:** TipTap (редактор с кастомными нодами).

**Backend & Cloud:**
- **Database:** Supabase (PostgreSQL) — хранение данных, связей и настроек.
- **Storage:** Cloudinary — хранение медиафайлов без лимитов Supabase.
- **Logic:** PostgreSQL RPC для оптимизированной сортировки массивов данных.
